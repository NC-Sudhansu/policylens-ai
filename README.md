AI-powered insurance policy analyzer built for the Indian market.

## What it does
- Upload any insurance policy PDF
- Get plain English summary instantly
- Email summary as PDF
- Find better alternatives in Indian market
- Generate detailed quotes
- Chat with AI agent if you don't have documents

## Tech Stack
- Python + Streamlit
- Groq API (LLaMA 3.3 70B)
- PyMuPDF + ReportLab
- Gmail SMTP

## Live Demo
https://policylens-ai.streamlit.app/

## Run Locally
1. Clone this repo
2. pip install -r project1-policy-summarizer/requirements.txt
3. Create .env file with your keys
4. streamlit run project1-policy-summarizer/app.py
